<h1 align="center">📇 Smart Contact Manager (SCM)</h1>

<p align="center">
A full-stack web application built using <b>Spring Boot</b> that allows users to securely manage their personal contacts with authentication and database integration.
</p>

---

## 🚀 Features

- 🔐 Secure authentication & authorization using <b>Spring Security</b>  
- ➕ Add, update, and delete contacts (CRUD operations)  
- 📋 Manage and view contact list  
- 🔍 Search contacts by name or email  
- 🛡 Secure session handling  
- 📱 Responsive UI using <b>Thymeleaf</b> and <b>Bootstrap</b>

## 🏗 Architecture

Follows <b>MVC (Model-View-Controller)</b> architecture:

- 🎯 Controller → Handles incoming requests  
- ⚙️ Service → Business logic  
- 🗄 Repository → Database interaction using JPA/Hibernate  
- 🎨 View → Thymeleaf templates
